# Feature Goal Documentation Schema (konceptuell konfiguration)

Denna fil dokumenterar vilka sektioner (DocSectionId) som är relevanta
för Feature Goal-dokumentation. Den speglar FEATURE_GOAL_DOC_SCHEMA i
src/lib/documentationTemplates.ts och är tänkt som underlag för en
framtida schema-driven renderingsmotor.

Aktuella sektioner:

- title-metadata
- summary
- scope
- epics-overview
- flow
- dependencies
- business-scenarios
- test-linking
- related-items
