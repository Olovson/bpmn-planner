# Business Rule / DMN Documentation Schema (konceptuell konfiguration)

Denna fil dokumenterar vilka sektioner (DocSectionId) som är relevanta
för Business Rule / DMN-dokumentation. Den speglar
BUSINESS_RULE_DOC_SCHEMA i src/lib/documentationTemplates.ts.

Aktuella sektioner:

- title-metadata
- summary
- inputs
- decision-logic
- outputs
- business-rules-policy
- business-scenarios
- test-linking
- related-items
