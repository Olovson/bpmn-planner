# Epic Documentation Schema (konceptuell konfiguration)

Denna fil dokumenterar vilka sektioner (DocSectionId) som är relevanta
för Epic-dokumentation (User Task / Service Task). Den speglar
EPIC_DOC_SCHEMA i src/lib/documentationTemplates.ts.

Aktuella sektioner:

- title-metadata
- summary
- inputs
- flow
- outputs
- business-rules-policy
- business-scenarios
- test-linking
- related-items
