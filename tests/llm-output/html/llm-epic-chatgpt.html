Syfte & Effekt

Epiken 'Approve Application' hanterar godkännandeprocessen för låneansökningar. Den spelar en central roll i kreditbedömningsprocessen genom att säkerställa att alla nödvändiga kontroller och beslut har gjorts innan ett lån beviljas. Detta är en User Task där handläggaren granskar och godkänner ansökningar baserat på systemets rekommendationer.

Inputs

Input Källa Format Validering
Fält: Ansöknings-ID; Datakälla: Intern databas; Typ: Sträng; Obligatoriskt: Ja; Validering: Kontrollera att ID finns; Felhantering: Returnera felmeddelande vid ogiltigt ID.
Fält: Kreditvärdighetspoäng; Datakälla: Extern kreditbyrå; Typ: Numerisk; Obligatoriskt: Ja; Validering: Kontrollera att poängen är över 600 (exempelvärde); Felhantering: Avvisa ansökan vid låg poäng.
Fält: Inkomstuppgifter; Datakälla: Intern databas; Typ: Numerisk; Obligatoriskt: Ja; Validering: Kontrollera mot deklarerade uppgifter; Felhantering: Begär kompletterande uppgifter vid avvikelse.
Fält: Belåningsgrad; Datakälla: Intern databas; Typ: Numerisk; Obligatoriskt: Ja; Validering: Kontrollera att belåningsgraden är under 85 % (exempelvärde); Felhantering: Avvisa ansökan vid hög belåningsgrad.

Funktionellt flöde

  1. Handläggaren loggar in i systemet och navigerar till godkännandeprocessen.
  2. Systemet hämtar och visar all relevant information om ansökan.
  3. Handläggaren granskar ansökan och gör en bedömning baserad på tillgängliga data.
  4. Systemet registrerar handläggarens beslut och uppdaterar ansökningsstatusen.
  5. Vid behov skickas ansökan tillbaka för komplettering eller vidare granskning.
  6. Handläggaren avslutar processen genom att godkänna eller avslå ansökan.

Output

  • Systemet tar emot ansökningsdata från interna och externa källor.
  • Beslutsdata skickas vidare till bankens centrala datalager för arkivering.
  • Konsumenter av data inkluderar riskavdelningen och kundtjänst.

Affärsregler som triggas

  • Epiken stödjer bankens kreditpolicy genom att säkerställa att alla ansökningar följer fastställda regler.
  • Bedömningar av skuldkvot och belåningsgrad genomförs automatiskt enligt policy.
  • Epiken säkerställer att AML/KYC-regler efterlevs genom att integrera med relevanta system.
  • Systemet tillåter handläggare att överstyra automatiska beslut vid behov, i enlighet med bankens riktlinjer.
