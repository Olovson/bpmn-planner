Sammanfattning & scope

Målet med denna funktion är att effektivisera insamlingen av intern data för kreditbedömning av bolånekunder. Genom att automatisera datainsamlingen kan vi förbättra precisionen i kreditbeslut och minska handläggningstiden. Detta stödjer bankens strategi att erbjuda snabbare och mer tillförlitliga kreditbeslut, vilket förbättrar kundupplevelsen och minskar risken för felaktiga bedömningar.

Omfattning & Avgränsningar

  • Ingår: Ingår: insamling av intern kund- och engagemangsdata för kreditbedömning.; Ingår: automatisering av datainsamlingsprocessen för att minska handläggningstiden.; Ingår: integration med befintliga interna system för att säkerställa datakonsistens.; Ingår: förbättrad precision i kreditbeslut genom användning av uppdaterad intern data.
  • Ingår inte: Ingår inte: eftermarknadsprocesser och generella engagemangsändringar.; Ingår inte: externa datakällor eller tredjepartsintegrationer.

Affärsflöde

  1. Kunden initierar en ansökan om bolån via bankens webbportal.
  2. Systemet börjar automatiskt samla in relevant intern data för kreditbedömning.
  3. Den insamlade datan valideras mot befintliga regler och kriterier.
  4. Ett preliminärt kreditbeslut genereras baserat på den validerade datan.
  5. Kunden informeras om det preliminära beslutet och nästa steg i processen.

Kritiska beroenden

  • Beroende: Interna system; Id: kunddatabas; Beskrivning: används för att hämta aktuell kundinformation.
  • Beroende: Regelmotor/DMN; Id: kreditvärdighetsbedömning; Beskrivning: används för att fatta preliminära och slutliga kreditbeslut.
  • Beroende: API-integration; Id: datavalidering; Beskrivning: säkerställer att all insamlad data är korrekt och aktuell.

Affärs-scenarion

Scenario Beskrivning Typ (Happy/Edge/Error) Förväntat resultat
S1 Successful Data Gathering Happy All nödvändig data samlas in och valideras korrekt, vilket leder till ett preliminärt godkännande.
S2 Data Validation Error Error Fel upptäcks i den insamlade datan, vilket kräver manuell granskning innan ett beslut kan fattas.
S3 Incomplete Data Edge Viss data saknas, vilket fördröjer processen tills kompletterande information kan erhållas.
