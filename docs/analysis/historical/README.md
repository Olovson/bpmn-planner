# Historiska Analyser

Denna mapp innehåller historiska analyser och temporära dokument som skapades under utveckling.

## Innehåll

Dessa dokument är arkiverade för referens men behöver normalt inte läsas:

- **ANALYSIS-filer** - Temporära analyser från utvecklingsprocessen
- **VERSIONING-filer** - Historiska versioning-analyser (se `architecture/VERSIONING_FINAL_DECISION.md` för aktuell info)
- **TEMPLATE_CONTENT-filer** - Historiska template-analyser
- **STRATEGY-filer** - Historiska strategidokument

## När ska du läsa dessa?

Normalt **aldrig** - dessa är historiska dokument. Om du behöver förstå bakgrunden till ett beslut, kolla huvuddokumentationen först.
