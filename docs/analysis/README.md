# Analysis Documentation

Denna mapp innehåller analyser och dokumentation av olika aspekter av BPMN Planner.

## Struktur

```
analysis/
├── README.md (denna fil)
└── [aktuella analyser]  # Analyser i root-nivå som är relevanta och aktuella
```

**Uppdaterad 2025-01-XX:** Gamla completed/, historical/, temporary/ och current/ mappar har tagits bort för att minska dokumentationsmängden.

## Kategorier

### Aktuella Analyser (root-nivå)
Analyser som kan vara relevanta, men inte alltid uppdaterade, t.ex.:
- `TESTINFO_GENERATION_COMPLETE_ANALYSIS.md` - Komplett analys av testinfo-generering
- `TESTINFO_VS_DOCUMENTATION_VALIDATION.md` - Validering av testinfo vs dokumentation
- `CONSOLIDATION_PLAN.md` - Plan för konsolidering av storage paths
- `BATCH_UPLOAD_AND_GENERATION_LOGIC.md` - Logik för batch-uppladdning
- `EXACT_GENERATION_ORDER.md` - Exakt genereringsordning
- `BPMN_FILE_MANAGER_SIZE_ANALYSIS.md` - Analys av filstorlekar
- `CODE_REFACTORING_PLAN.md` - Plan för kodrefaktorering

## När ska du läsa dessa?

- **Analyser** – När du behöver historik eller resonemang bakom beslut. Verifiera alltid mot koden.
- Se [`../guides/user/`](../guides/user/) för användarguider
