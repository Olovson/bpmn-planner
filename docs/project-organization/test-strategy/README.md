# Teststrategi

Plats för dokumentation om teststrategi, testmetoder och testprocesser.

## Vad ska här?

- Teststrategi dokument
- Testmetoder (unit, integration, e2e, etc.)
- Testprocesser
- Testdata management
- Test automation strategy
- Quality gates
- Test metrics och reporting

## Integration med BPMN Planner

BPMN Planner genererar test scenarios och scripts. Denna teststrategi definierar:
- Hur dessa ska användas
- När och var de ska köras
- Vilka roller som ansvarar
- Hur de kompletteras med manuell testning

## Lägg till dina dokument här

När du delar dina teststrategi-dokument kommer jag att:
1. Analysera och strukturera dem
2. Länka till BPMN Planner's test-generering
3. Skapa en sammanhållen testplan
4. Definiera integration mellan automatiserad och manuell testning
