# Projektstruktur

Plats för dokumentation om projektstruktur, organisation och arkitektur.

## Vad ska här?

- Projektorganisation
- Team structure
- Arkitekturöversikt
- Teknisk stack
- Integration points
- Deployment process
- Miljöer (dev, test, prod)

## Integration med BPMN Planner

BPMN Planner är en del av projektstrukturen och integrerar med:
- BPMN-filer (processdefinitioner)
- Testmiljöer (export-ready scripts)
- Dokumentationssystem
- CI/CD pipelines

## Lägg till dina dokument här

När du delar dina projektstruktur-dokument kommer jag att:
1. Analysera projektorganisationen
2. Mappa BPMN Planner till projektstrukturen
3. Definiera integration points
4. Skapa en sammanhållen översikt
