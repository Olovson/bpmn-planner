# Ways of Working

Plats för dokumentation om ways of working, best practices och processer.

## Vad ska här?

- Development process
- Code review process
- Git workflow
- Sprint/iteration process
- Communication guidelines
- Definition of Done
- Andra processdokument

## Lägg till dina dokument här

När du delar dina ways of working-dokument kommer jag att:
1. Läsa och analysera dem
2. Skapa strukturerade sammanfattningar
3. Länka till relevanta delar av BPMN Planner
4. Integrera med teststrategi och roller
