# Mallar för Framtida Projekt

Plats för mallar och strukturer som kan återanvändas i framtida projekt.

## Vad ska här?

- Projektstart-mallar
- Ways of working-mallar
- Teststrategi-mallar
- Rollbeskrivningar-mallar
- Checklistor
- Best practices sammanfattningar

## Syfte

När vi har dokumenterat och strukturerat detta projekt kan vi:
1. Extrahera mallar och best practices
2. Skapa återanvändbara strukturer
3. Definiera en standard för framtida projekt
4. Säkra att kunskap bevaras och återanvänds

## När detta skapas

Mallarna skapas efter att vi har:
1. Dokumenterat ways of working
2. Definierat teststrategi
3. Beskrivit roller och ansvar
4. Skapat en total projektplan
