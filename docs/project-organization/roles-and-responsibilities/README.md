# Roller och Ansvar

Plats för dokumentation om roller, ansvar och förväntningar.

## Vad ska här?

- Rollbeskrivningar
- Ansvar och förväntningar
- Beslutsprocesser
- Escalation paths
- Team structure
- Communication matrix

## Integration med BPMN Planner

BPMN Planner används av olika roller:
- **Test Lead**: Använder genererade test scenarios och scripts
- **Developers**: Använder dokumentation och BPMN-struktur
- **Product Owners**: Använder processöversikt och dokumentation
- **Architects**: Använder hierarki och dependency tracking

## Lägg till dina dokument här

När du delar dina roller-dokument kommer jag att:
1. Mappa roller till BPMN Planner features
2. Definiera ansvar för varje roll
3. Skapa workflows för olika roller
4. Integrera med ways of working och teststrategi
