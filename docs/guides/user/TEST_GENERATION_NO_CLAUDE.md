# Testgenerering utan LLM

Om LLM inte är aktiverat:

- Inga E2E‑scenarier genereras.
- Feature Goal‑scenarier genereras inte (resultatet blir tomt).

Du kan fortfarande:
- Generera dokumentation (mallar + strukturell info).
- Köra process‑ och grafanalys i UI.

För fullständig logik: `docs/testing/TEST_GENERATION.md`.
